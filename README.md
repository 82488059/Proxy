#### 软件架构
软件架构说明


#### 安装教程
在Proxy目录下配置conf.py(与db_public.py同级)
```py
#!/usr/bin/python3
# -*- coding: utf-8 -*-
db_charset = 'utf8'
db_public = 'dbname'
db_public_host = '127.0.0.1'
db_public_user_name = 'root'
db_public_user_pwd = 'pwd'
db_port = 3306
```

#### 使用说明

运行RunProxy.py
